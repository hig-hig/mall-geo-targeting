# モール基本情報の受け入れ

実モールの基本情報とmetadataを配置する用途です。現在の`config/malls.yaml`は模擬設定であり、実在モールのデータは同梱していません。命名・必須項目は`data/raw/README.md`に従ってください。

## 現在の対象施設

- 対象：イオンモールむさし村山
- 初期分析半径：10,000m
- OSM・商業POIに必要な取得範囲：中心から11,000m以上（分析半径＋buffer 1,000m）
- 中心座標：`35.7465, 139.3850`（暫定。OSMまたは公式地図で再確認）
- 分析用規模：総賃貸面積約78,000㎡
- attractiveness：`1.0`（暫定初期値）

## 競合候補台帳と登録条件

確認済みの施設情報と未確認項目は`competitor_candidates.yaml`へ記録します。次の条件がすべて揃うまで、本番`competitor_malls`へ登録しません。

- 公式または信頼できる住所
- 地図上で確認した中心座標
- 分析用床面積（原則として総賃貸面積）
- 情報源URLと取得日
- 暫定項目の明示

現在、ららぽーと立川立飛とイオンモール日の出は公式施設規模を確認済みですが、中心座標が未確認です。モリタウンは統一的な公式総賃貸面積と中心座標が未確認です。候補ごとに施設境界、建物中心、主要入口のどれを中心点とするかを統一して確認してください。

attractivenessの初期参考値は、対象を`1.0`として`競合の総賃貸面積 / 対象の総賃貸面積`で計算します。ららぽーと立川立飛は約`0.808`、イオンモール日の出は約`0.821`です。これは床面積だけの暫定モデルであり、ブランド力、テナント構成、駐車場、駅直結性、来館実績は反映しません。座標確認と業務レビューが終わるまで、本番設定には適用しません。
