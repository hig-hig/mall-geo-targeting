# モール基本情報の受け入れ

実モールの基本情報とmetadataを配置する用途です。現在の`config/malls.yaml`は模擬設定であり、実在モールのデータは同梱していません。命名・必須項目は`data/raw/README.md`に従ってください。

## 現在の対象施設

- 対象：イオンモールむさし村山
- 初期分析半径：10,000m
- OSM・商業POIに必要な取得範囲：中心から11,000m以上（分析半径＋buffer 1,000m）
- 中心座標：`35.746390, 139.384750`（公式アクセス情報とOSM建物Polygonを照合済み）
- 分析用規模：総賃貸面積約78,000㎡
- attractiveness：`1.0`（全施設共通で非規模補正を適用しない正式仕様）

## 競合候補台帳と登録条件

確認済みの施設情報と未確認項目は`competitor_candidates.yaml`へ記録します。次の条件がすべて揃うまで、本番`competitor_malls`へ登録しません。

- 公式または信頼できる住所
- 地図上で確認した中心座標
- 分析用床面積（原則として総賃貸面積）
- 情報源URLと取得日
- 暫定項目の明示

ららぽーと立川立飛、イオンモール日の出、モリタウンを競合設定へ登録しています。モリタウンの中心座標は公式アクセス情報とOSM retail Polygonを照合した代表点です。旧住所`田中町562-1`と現住所`代官山2-3-1`は、昭島市の2024年8月の住居表示実施による新旧表記として候補台帳へ併記しています。

Huffの規模項には施設規模を一度だけ使い、`attractiveness=1.0`を全施設へ共通適用します。これは非規模補正を適用しない正式仕様で、対象施設基準のGLA比は使用しません。モリタウンの59,747㎡は公表された「店舗面積」であり、厳密な国際的GLAではありませんが、現行モデルではGLA相当値として使用します。施設範囲はモリタウン本体とし、MOVIX昭島、モリパーク アウトドアヴィレッジ、ニトリ、スポーツデポ等は別のHuff競合にしません。これら周辺施設がOSM商業POIへ含まれることによる概念的な二重評価リスクは、metadataに記録しています。
